Testing Instructions

  1. Load this page in a browser with the Creator OS extension installed
  2. Ensure this page's URL matches a pattern in your whitelist
  3. Open the extension popup and verify the hostname and toggle state
  4. Check the browser console for capture logs (if DEBUG=true)
  5. Use "Capture Now" to trigger a manual capture
  6. Verify that the captured DOM excludes:
    • Script tags and their content
    • Style tags and their content
    • Event handler attributes (onclick, onmouseover, etc.)
    • Form input values
    • Selected option states
    • Iframe elements
    • Noscript elements
    • Preload/prefetch link elements
